> This project was heavily "vibecoded" with AI. I put it together quickly because I just wanted a simple deezer TUI for my own personal use.

## Controls
> Arrow Keys - Navigate
> 
> TAB - Switch Focus
> 
> Enter - Select
> 
> P - Play/Pause
> 
> / - Search
>
> Q - Quit
